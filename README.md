# PARSE FUNC
C library to map integer sets to functions and define basic mathematical constants

# INSTALL
```
cd
git clone https://github.com/CodeFromKerala/parse_func
cd parse_func
sudo make install
make clean
```

or just install SimplyCalc @ [SimplyCalc](https://github.com/CodeFromKerala/SimplyCalc)

# UNINSTALL

```
sudo make uninstall
```

# WHY

I made this library just to serve as a dependency to my project [SimplyCalc](https://github.com/CodeFromKerala/SimplyCalc)
But I figured why not embrace FOSS.
Hopefully youll use it without any problems.
If any arise just lmk.
